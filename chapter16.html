Summary — luận điểm chính
▼

SPSS không tự biết suy nghĩ. Bạn phải dạy nó hiểu các con số qua màn hình Variable Viewer trước khi chạy bất kỳ phép tính nào.

  • ①2 màn hình cốt lõi: Data Viewer (nhập số liệu như Excel) + Variable Viewer (khai báo tên, nhãn, Value Labels, Missing Values).
  • ②2 lệnh "chế biến" thần thánh: Recode (gộm nhóm dữ liệu — Tuổi cụ thể → Nhóm tuổi) và Compute (tạo biến mới từ công thức — Tổng điểm thang Likert).
  • ③Menu Analyze — tất cả trong 1: Frequencies (tần suất), Crosstabs (bảng chéo + Chi-square), Correlate (Pearson/Spearman), Compare Means (Eta).
  • ④Lời khuyên xương máu: Mở SPSS trước, lập sẵn Variable Viewer, chạy thử 10 mẫu. Nếu Crosstabs không ra như ý → sửa ngay bảng hỏi.

16.1Data Viewer, Variable Viewer, Value Labels▶
📋 Data Viewer vs Variable Viewer
Data Viewer = bảng tính chứa số liệu thô. Variable Viewer = nơi “dạy” SPSS hiểu các cột đó: tên biến, nhãn, kiểu dữ liệu.
Value Labels
Gán chữ cho số:
1 = Nam, 2 = Nữ
Không gán → ra kết quả số 1,2,3 — không hiểu gì!
Missing Values
Khai báo số đại diện “để trống” (VD: 99)
SPSS sẽ loại khi tính Mean → tránh sai lệch.
16.2Recode và Compute — Chế biến dữ liệu▶
🔄 RECODE — Gộm nhóm dữ liệu

Biến tuổi 18, 19, 20... dùng Recode into Different Variables gộm thành Nhóm 1 (18-25), Nhóm 2 (26-35). Sau đó dùng cho Crosstabs.

Transform → Recode into Different Variables → Old and New Values
🧮 COMPUTE — Tạo biến mới

Cộng 5 câu Likert chia 5 → biến DongLuc_TrungBinh. Hoặc cộng thời gian tập các máy: totalmin = cardmins + weimins + othmins.

Transform → Compute Variable → Nhập công thức toán học
16.3Menu Analyze — Chạy tất cả▶
Mục tiêu Đường dẫn SPSS Kết quả cho
Bao nhiêu % chọn?Analyze → Descriptive Statistics → FrequenciesNominal / Ordinal
So sánh Nam vs NữAnalyze → Descriptive Statistics → CrosstabsNominal × Nominal
Tương quan 2 biến sốAnalyze → Correlate → BivariatePearson / Spearman
Trung bình theo nhómAnalyze → Compare MeansInterval + Nominal (Eta)
⚠️ SPSS ném ra rất nhiều bảng biểu dài ngoẵng. Việc của bạn không phải copy toàn bộ vào luận văn, mà là đọc hiểu, chắt lọc — đặc biệt chú ý chỉ số Sig. / p-value.
16.4Graphs, File Management, Transferable Skills▶
📋 Graphs — Chart Builder
Dùng Graphs → Chart Builder để vẽ Scatterplots (biểu đồ phân tán) quan sát xu hướng trước khi chạy Pearson. Nhìn biểu đồ trước = tránh chạy sai công cụ.
💾 Lưu và quản lý file
Lưu (Save) liên tục! File .sav là dữ liệu đã khai báo. File .spv là kết quả Output. Quản lý file khoa học: không để "data_final_v5_CUOI.sav".
🔄 Transferable skill
Kỹ năng SPSS (hoặc R, Python, Stata tương tự) là lợi thế lớn. Lệnh Compute = Pivot Table / Power Query trong Excel. Lệnh Recode = phân khúc khách hàng.

📋 Case Study: Phân tích hài lòng NV TechZ (3 bước SPSS)
Bước 1 — Thiết lập: Variable Viewer: GioiTinh (Value 1=Nam, 2=Nữ). HaiLong: ai không điền → nhập 99, khai báo Missing = 99.
Bước 2 — Recode: Sếp muốn xem Gen Z vs Gen Y → Recode: tuổi 20-26 → Nhóm 1, 27-40 → Nhóm 2.
Bước 3 — Analyze: Correlate (Pearson) giữa Lương ↔ Hài lòng. Crosstabs: Gen Z/Y × tỉ lệ “Rất không hài lòng”.
💡 Đề xuất ứng dụng
Luận vănKhông để đến khi thu xong 500 mẫu mới mở SPSS. Lập sẵn Variable Viewer, chạy thử 10 mẫu, kiểm tra ngay.
Data AnalystNắm vững Compute để tính tổng/trung bình KPI, và Recode để phân khúc khách hàng VIP/Thường — xử lý hàng vạn dòng trong vài giây.
Đời sốngTư duy logic SPSS (Data thô → Khai báo luật chơi → Làm sạch → Phân tích) giúp cấu trúc bất kỳ bảng Excel/tài chính cá nhân của bạn.
